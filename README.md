# 💰お年玉運用アプリ💰

GMOあおぞらネット銀行の外部API（sunabar）を利用し、  
お年玉という「まとまったお金」を題材に、  
**お金の流れを可視化しながら学べるWebアプリ**を開発しました。

本アプリは、銀行API連携の基礎理解と、  
フロントエンド・バックエンド分離構成の実践を目的とした学習用アプリです。

---

## 📌 開発背景

お年玉は一度に大きな金額が手に入る一方で、

- 子ども自身が「いくら持っているか」を把握しづらい  
- 使い道を考える前に消費してしまう  

といった課題があります。

一方、親側も  
「細かく管理しすぎず、でも学びにはつなげたい」  
というニーズを抱えています。

そこで、  
**お金の動きを“体験として理解できる仕組み”**を目指し、  
銀行APIを活用したアプリを企画・開発しました。

---

## 🧩 主な機能

- **残高照会機能**  
  GMOあおぞらネット銀行APIを利用し、口座残高を取得・表示します。

- **口座一覧照会機能**  
  円普通預金口座およびつかいわけ口座の一覧を表示します。

- **入出金明細照会機能**  
  指定口座の入出金履歴を取得し、一覧で確認できます。

- **つかいわけ口座間振替機能**  
  つかいわけ口座間で金額を指定して振替を行います。

---

## 🔐 セキュリティ・設計で意識した点

- 銀行APIは **必ずバックエンド経由で呼び出し**
- APIキーは `.env` にて管理し、Git管理外とする
- フロントから送信される値はバックエンドでバリデーションを実施
- 外部APIエラー時もアプリが落ちないよう例外処理を実装
